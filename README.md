# Matjournal

En Android-fokuserad Expo/React Native-app för att registrera mat, dryck, läkemedel och symtom.

## Kom igång

```bash
npm install
npm run start
```

Starta sedan appen på Android med:

```bash
npm run android
```

## Kontrollera projektet

```bash
npm test
npm run lint
```

Appen använder Expo SDK 54. För EAS Build och felsökningskommandon, se [shell-kommandon](docs/shell-commands.md).
